使用說明

  • 獲取 Bot Token：與 @BotFather 對話創建機器人並獲取 Token
  • 頻道 ID：將機器人添加為頻道管理員後，頻道 ID 通常為負數（如 -100123456789）
  • 照片連結：必須是可公開訪問的 HTTPS 圖片 URL
  • 點擊生成的鏈接將直接向 Telegram 發送消息
  • 如需測試，請確保機器人有權限發送到目標頻道

使用说明

  • 获取 Bot Token：与 @BotFather 对话创建机器人并获取 Token
  • 频道 ID：将机器人添加为频道管理员后，频道 ID 通常为负数（如 -100123456789）
  • 图片链接：必须是可公开访问的 HTTPS 图片 URL
  • Base64 编码：上传图片后会自动生成
  • 点击生成的链接将直接向 Telegram 发送消息
  • 如需测试，请确保机器有权限发送到目标频道
